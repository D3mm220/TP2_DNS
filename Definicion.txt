Domain Name System o mejor conocido como DNS, es un sistema de nomenclatura que se encarga de traducir los nombres que tipeamos para que sean traducidos en direcciones IP, para comunicarse dentro de la red.
	Su función es la siguiente: Si tratamos de conectarnos a una página desde nuestra computadora, lo primero que nuestro dispositivo va a hacer es fijarse si en la caché DNS local ya se encuentra el dominio traducido (dirección IP) por si anteriormente el dispositivo se había conectado con esa dirección. Si no se encuentra la caché, el próximo paso es pedirle al resolver (servidor dns local) que tiene conexión con el dns del ISP (Internet Service Provider). El dispositivo le pide al resolver que le consiga la dirección IP de la página a la que quiere acceder y el resolver se fija
